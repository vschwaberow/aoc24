- CMake 3.26+
- C++26 capable compiler (GCC 13+ or Clang 16+)

## Quick Start

```bash
git clone https://github.com/vschwaberow/aoc24
cd aoc24
mkdir build && cd build
cmake ..
cmake --build .
```
